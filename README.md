Lucide React(1.7.0) / ESLint(9.39.4)

# Backend
CORS(2.8.6) / Express(5.2.1)

# Database
SQLite

# Infrastructure/Deployment
Node.js / 로컬 개발 환경 기반


# 사용 이유
React + Vite
컴포넌트 기반 UI + 빠른 HMR로 개발 생산성 극대화, 수정 시 즉시 반영되어 개발 속도에 유리함.

React Router
페이지 이동 시 새로고침 없이 부드럽게 전환되도록 처리

Axios
서버와 데이터를 주고받는 통신 도구. 
오류 처리가 편리함.    

Lucide React
가볍고 깔끔한 아이콘 모음. 크기,색상 조절이 쉬움

CORS
프론트-백 간 크로스 오리진 통신 보안 정책 처리

SQLite
별도 설치 없이 파일 하나로 동작하는 
초경량 데이터 저장소. 
시간표, 할 일 목록 등의 데이터를 관리

Node.js + Express
프론트와 동일 언어(JS)로 풀스택 생산성 확보.
비동기 I/O 기반 고속 응답
